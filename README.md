# Descripción
-------------------------------
En este desafio llamado  Alura Store tuve que ayudar a José, un emprendedor con 4 tiendas, a decidir cuál debería cerrar para poer invertir en un nuevo negocio.

 El reto consistía en analizar los datos de cada tienda y encontrar cuál tenía el menor desempeño general.     Para eso, trabajé con informcaión relacionada con el ingreso total, el número de ventas, el producto más       vendido y el menos vendido, la calificación de cada tienda y el nivel estimado de producción. A partir de esos datos, realicé procedimientos y visualizaciones que me permitieron comparar los resultado de  cada tienda.
 
---------------------------------
## Objetivos:
  • Identificar cuál de las tiendas generaba menos ganancais o mostrara menor potencial de crecimiento.
  
  • Aplicar técnicas básicas de ánalisis estadístico para interpretar correctamente la información obtenida.
  
  • Elaborar gráficos y visualizaciones que permitan observar las diferencias ente las tiendas de forma clara y comprensible.
  
• Desarrollar un conclusión lógica y argumentada que ayude a tomar la mejor decisión.

• Interpretar los resultados obtenidos a partir de ánalisis gráfico y numérico de los datos.

--------------------------------------
## Metodologías usadas en Google Colab
1. Análisis estadístico descriptivo:
    - Uso de librerías como pandas para la manipulación de datos y numoy para cálculos numéricos.
2. Visualización de datos:

     - Creación de gráficos(barras, líneas, histogramas, etc.) con librerías como matplotlib y seaborn para
        dectar patrones y comprar resultados entre tiendas.
3. Interpretación y comparación:
  
      - Análisis conjunto de las visualizaciones y estadísticas para identificar cuál tienda tiene menor.             desempeño
