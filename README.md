# Analisis de ventas y redimiento de las Tiendas
  Descripción
-------------------------------
En este desafio llamdo  Alura Store tuve que ayudar a José, un emprendedor con 4 tiendas, a decidir cuál debería cerrar para poer invertir en un nuevo negocio.

 El reto consistía en analizar los datos de cada tienda y encontrar cuál tenía el menor desempeño general.     Para eso, trabajé con informcaión relacionada con el ingreso total, el número de ventas, el producto más       vendido y el menos vendido, la calificación de cada tienda y el nivel estimado de producción
 A partir de esos datos, realicé procedimientod y visualizaciones que me permitieron compara los resultado de  cada tienda.
 
---------------------------------
## Objetivos:
1. Aplicar técnicas básicas de ánalisis estadístico para interpretar correctamente la información obtenida
2. Elaborar gráficos y visualizaciones que permitan observar las diferencias ente las tiendas de forma clara y comprensible
3. Desarrollar un conclusión lógica y argumentada que ayude a tomar la mejor decisión
